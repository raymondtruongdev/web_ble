Bluetooth Low Energy (BLE)

Khám phá sức mạnh của Web Bluetooth API: Kết nối các thiết bị phần cứng trực tiếp từ trình duyệt web của bạn.

1

Kiến trúc GATT

Cấu trúc phân tầng dữ liệu
  • → Profile: Tập hợp Services (VD: Heart Rate Monitor).
  • → Service: Nhóm dữ liệu logic (VD: Pin, Cảm biến).
  • → Characteristic: Đơn vị nhỏ nhất, chứa giá trị và quyền.
2

Quét & Kết nối

Yêu cầu bảo mật

Web Bluetooth yêu cầu kết nối bảo mật HTTPS và tương tác trực tiếp từ người dùng (User Gesture) để kích hoạt hộp thoại.

3

Truy xuất Dữ liệu

Thao tác Characteristic

Sử dụng DataView để xử lý các mảng byte nhị phân qua Bluetooth.

Read: Đọc giá trị tức thời.
Notify: Cập nhật tự động.
ReadValue.js
const service = await server.getPrimaryService('battery_service');
const char = await service.getCharacteristic('battery_level');
const value = await char.readValue();
console.log(`Pin còn: ${value.getUint8(0)}%`);

Các phương thức truyền tin

Phương thức Đặc điểm Ứng dụng
Read Yêu cầu từ Client Thông tin thiết bị, pin.
Write Gửi lệnh từ Client Bật/tắt LED, cấu hình.
Notify Tự động đẩy dữ liệu Theo dõi nhịp tim.
