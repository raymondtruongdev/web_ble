Tương tác dữ liệu BLE

Làm chủ các thao tác Read, Write và cơ chế Stream dữ liệu thời gian thực.

1

Đọc dữ liệu (Read)

Truy vấn trạng thái

Phương thức readValue() lấy giá trị hiện tại của một Characteristic dưới dạng DataView.

  • ✔ Thích hợp cho dữ liệu tĩnh: Tên, Pin, Version.
  • ✔ Cần lưu ý định dạng Byte (Endianness).
2

Gửi lệnh (Write)

Điều khiển thiết bị

Gửi dữ liệu xuống phần cứng có xác nhận hoặc không xác nhận.

Mẹo: writeValueWithResponse đảm bảo gói tin không bị mất.
3

Dòng dữ liệu (Stream)

Cơ chế Notify

Nhận thông báo tự động khi cảm biến có giá trị mới thay vì thăm dò liên tục.

1. startNotifications()
2. Listen characteristicvaluechanged
Stream.js
await characteristic.startNotifications();

characteristic.addEventListener('characteristicvaluechanged', (e) => {
  const value = e.target.value;
  const heartRate = value.getUint8(1);
  updateUI(heartRate);
});
4

Xử lý gói tin

Trích xuất các bit dữ liệu từ gói tin phức hợp để tiết kiệm năng lượng.

Kỹ thuật Parsing

Sử dụng DataView (getUint16, getFloat32) với littleEndian = true.

Parser.js
function handleData(event) {
  const view = event.target.value;
  const status = view.getUint8(0);
  const temp = view.getInt16(1, true) / 100;
  console.log(status, temp);
}
